🔹 1. Система управления пользователями и сессиями
1.1. Создание и вход
Открытие сайта → сразу чистый холст (без регистрации)

Быстрая регистрация (email + пароль) / вход через Google, Discord

Анонимный вход по ссылке-приглашению (без аккаунта)

Роли: Мастер (GM), Игрок (Player), Наблюдатель (Observer)

Переключение ролей внутри комнаты (только GM может дать права)

Сохранение истории сессий (для зарегистрированных)

Личный кабинет: аватар, имя, настройки приватности

1.2. Комнаты (сессии)
Создание комнаты в 1 клик (автоматическая генерация короткой ссылки)

Приглашение по ссылке (с возможностью установить пароль)

Ограничение срока жизни комнаты (1 час, 1 день, бессрочно)

Закрытые комнаты (только по прямому приглашению)

Комнаты с очередью (ожидание одобрения GM)

Копирование комнаты (дублирование всех настроек, но без истории)

1.3. Настройки комнаты
Название комнаты, описание, обложка

Тип игры (D&D 5e, Pathfinder, Custom → выбор правил)

Права игроков: могут ли перемещать чужие токены, кидать кубики за всех, рисовать

Максимальное количество игроков (2–20)

Автоматическое удаление комнаты после N дней без активности

🔹 2. Холст и карта
2.1. Базовый холст
Бесконечное полотно (виртуальный размер 10 000×10 000 клеток)

Масштабирование: колесико мыши (от 5% до 5000%)

Перетаскивание холста (зажатая правая кнопка мыши или пробел + левая)

Привязка камеры к токену (опционально, для игроков)

Кнопка «Сбросить вид» (вернуться в центр)

2.2. Сетка и выравнивание
Типы сетки: квадратная, гексагональная (горизонтальная/вертикальная), изометрическая, без сетки

Размер клетки: настраиваемый (от 10 до 200 пикселей)

Цвет и прозрачность сетки

Автовыравнивание карты: пользователь выделяет 3×3 клетки → система подгоняет сетку

Смещение сетки вручную (клавиши ←↑↓→, shift+стрелка для микрошага)

2.3. Сцены (экраны)
Создание новой сцены (Ctrl+N)

Переключение между сценами (вкладки или выпадающий список)

Копирование сцены

Удаление сцены

Установка фонового изображения (карта) — перетаскиванием на холст

Установка нескольких фоновых слоёв (например, нижний слой — карта, верхний — полупрозрачная калька)

Параллакс-фон (движется медленнее основного холста) для атмосферы

2.4. Слои
Слой 0: Фон (карта, статичные объекты, не выделяемые)

Слой 1: Препятствия (стены, здания — для LOS)

Слой 2: Токены (игроки, монстры, объекты)

Слой 3: Рисунки (временные пометки, лазер)

Слой 4: Текст (надписи)

Слой 5: Освещение (динамические источники света)

Слой 6: Эффекты (дым, огонь, анимации погоды)

Видимость каждого слоя: показать всем / только GM / скрыть

🔹 3. Токены и объекты
3.1. Создание токенов
Из файла (PNG, WEBP, SVG, JPG) — drag&drop на холст

Из буфера обмена (Ctrl+V)

Из компендиума (перетаскивание монстра)

Из мастерской токенов (портрет + рамка → готовый токен)

Из URL (вставить ссылку на картинку)

Генерация простого цветного кружка с буквой (быстрый токен)

3.2. Редактирование токена
Выделение (клик ЛКМ) → рамка с квадратиками для ресайза

Множественный выбор (Ctrl+клик, рамка выделения)

Перемещение (перетаскивание) с привязкой к сетке или свободно (toggle)

Изменение размера (угловые маркеры) — пропорционально или нет

Вращение (верхний маркер) с шагом 15°, 45° или свободно

Зеркальное отражение (горизонтальное/вертикальное)

Изменение прозрачности (0–100%)

Настройка яркости, контрастности, цветового тона (фильтры)

Добавление текстовой метки (имя, HP) над токеном — стиль, цвет, размер шрифта

Установка рамки (цвет, толщина, скругление, стиль: пунктир, двойная)

3.3. Свойства токена (инспектор)
Имя (отображается над токеном)

Размер в клетках (1×1, 2×2, 3×3…)

HP текущее / максимальное

Temporary HP

AC (Класс брони)

Инициатива (число, редактируемое)

Скорость (клеток за ход)

Состояния: отравлен, парализован, невидим, в укрытии и т.д. (иконки над токеном)

Привязка к листу персонажа (из компендиума или ручной)

Примечания (только для GM)

3.4. Действия с токенами (контекстное меню ПКМ)
Копировать / Вставить / Удалить

Переместить в слой (верхний/нижний)

Сбросить поворот и размер

Показать карточку (статы, инвентарь)

Добавить в трекер инициативы

Нанести урон (открыть окно: ввести число, тип урона)

Исцелить (добавить HP)

Применить состояние (выбрать из списка)

Закрепить за игроком (владелец токена)

Заблокировать перемещение (только GM может двигать)

3.5. Зоны (области)
Создание прямоугольной или круглой зоны

Зона может быть: опасная (урон в конце хода), лечебная, магическая, триггерная (выдаёт сообщение)

Назначение цвета и названия зоны

Автоматическое применение эффекта при входе токена в зону

🔹 4. Инструменты рисования и измерения

4.2. Измерения
Линейка (от точки до точки) — показывает расстояние в клетках и реальных единицах (футы, метры, выбирается в настройках)

Измерение площади (прямоугольник, эллипс) — показывает количество клеток

Измерение пути (кликаешь точки, двойной клик завершает) — общая длина

Инструмент «прицел» — показывает расстояние до ближайшего токена при наведении

4.3. Вспомогательные инструменты
Лупа (увеличение области)

Пипетка (скопировать цвет с рисунка или токена)

Перекрестие (точное позиционирование)

Источники света: прикрепляемые к токенам или статичные (факел на стене)

Параметры источника: радиус (свет), тусклый свет, цвет, мигание

Глобальное освещение (дневной свет) — вкл/выкл

Режим «ночного зрения» для рас (увеличивает радиус видимости)

Предпросмотр для GM (показывает, что видит выбранный токен)

10.4. Скрытые действия
